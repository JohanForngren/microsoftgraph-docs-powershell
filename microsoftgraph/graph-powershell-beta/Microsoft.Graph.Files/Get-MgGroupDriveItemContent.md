# Get-MgGroupDriveItemContent

## SYNOPSIS
Get content for the navigation property items from groups

## SYNTAX

### Get1 (Default)
```
Get-MgGroupDriveItemContent -DriveId <String> -DriveItemId <String> -GroupId <String> -OutFile <String>
 [-PassThru] [<CommonParameters>]
```

### GetViaIdentity1
```
Get-MgGroupDriveItemContent -InputObject <IFilesIdentity> -OutFile <String> [-PassThru] [<CommonParameters>]
```

## DESCRIPTION
Get content for the navigation property items from groups
